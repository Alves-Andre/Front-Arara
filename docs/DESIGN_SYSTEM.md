# ARARA Design System

ARARA: Acompanhamento Remoto de Areas em Recuperacao Ambiental.

## Brand Assets

Logo files should live in `public/brand`.

Recommended names:

- `arara-logo.svg`
- `arara-logo-horizontal.svg`
- `arara-icon.svg`

## Colors

- Primary: `#0066CC`
- Secondary: `#FFD23F`
- Tertiary: `#00AEEF`
- Neutral: `#74777F`

## Typography

- Font family: Inter
- Headline, body, and label tokens inherit the global Tailwind font stack.

## Component Defaults

- Radius: `8px`
- Primary buttons use `primary`.
- Secondary buttons use `secondary`.
- Inputs, cards, badges, and map surfaces use the global CSS variables in `src/app/globals.css`.
